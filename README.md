# 💫 About Me:
I’m a Computer Science student at KIET Group of Institutions passionate about building scalable web applications and cloud-deployed systems, including a containerized multi-tier website hosted on AWS EC2 using Docker, Docker Compose, and Nginx. I enjoy transforming ideas into real products — from responsive React interfaces to reliable backend deployments — and I’m eager to collaborate on impactful full-stack MERN and cloud-native projects while strengthening my skills in backend architecture, DevOps workflows, and system design. Currently, I’m learning advanced React patterns, microservices, and production-level cloud deployment, and you can ask me about full-stack development, UI/UX design, AWS deployment, or containerized applications; fun fact: I’ve organized hackathons, led technical teams of 30+ members, and trained students in design tools, so I enjoy building communities as much as software.
